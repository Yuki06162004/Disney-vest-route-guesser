    GitHub と Firebase の始め方

    1. GitHub

    新しいリポジトリを作り、このフォルダのファイルをアップロードします。公開ページにするなら GitHub Pages で `planner.html` を開けます。

    2. Firebase

    Firebase コンソールで Web アプリを追加し、設定値を `firebase-config.js` に入れます。Firestore を有効化するとルーム共有が動きます。

    3. 公式待ち時間

    公式アプリの待ち時間を自動取得する公開APIは確認できないため、まずは手入力で運用します。提携できた場合だけ連携口を追加します。
